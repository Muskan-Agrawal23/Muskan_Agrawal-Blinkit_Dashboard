📊 Blinkit Data Analysis Project

This project is a data analysis and visualization study of Blinkit, focusing on understanding sales performance, customer behavior, and product trends using real or simulated dataset.

The objective of this project is to transform raw data into meaningful insights using Power BI dashboards.

🎯 Objectives:
Analyze overall sales and revenue trends
Identify top-performing products and categories
Understand city-wise and outlet-wise performance
Track customer purchasing patterns
🛠️ Tools Used:
Power BI
Excel (Data cleaning & preprocessing)
DAX (for calculated measures)
📌 Key Insights:
High-performing product categories contribute major revenue
Certain locations/outlets show higher sales efficiency
Seasonal trends affect overall demand patterns
🚀 Outcome:

This project helps in understanding real-world retail analytics and improves decision-making through data visualization.# Muskan_Agrawal-Blinkit_Dashboard
